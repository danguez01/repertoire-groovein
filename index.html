Connectez l'interface à votre Google Sheet via Apps Script (lecture + écriture en direct), ou en mode .xlsx (lecture seule).

⚡ Recommandé : Apps Script (sync bidirectionnel)

Mode alternatif : import .xlsx (lecture seule)
Apps Script — mise en place (~5 min, une seule fois) :
  1. Ouvrez votre Sheet → Extensions → Apps Script
  2. Collez le code de AppsScript_Code.gs dans Code.gs et sauvegardez
  3. Exécutez la fonction smokeTest et autorisez l'accès à votre Sheet
  4. Déployer → Nouveau déploiement → Application Web. Choisissez "Tout le monde" pour l'accès
  5. Copiez l'URL /exec dans le champ ci-dessus → "Connecter au Sheet"
Détails complets dans le fichier Instructions_AppsScript.md.
